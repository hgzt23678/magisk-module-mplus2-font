#### M Plus2 Font

日本語ユーザーに向けたM PLUS2フォントパッチです。
AndroPlus氏のモジュールが元となっております。

#### 重要

* このモジュールをインストールするには最新のMagiskを使用することをお勧めします。
  古いバージョンでインストールして何か起きても知りませんからね？
